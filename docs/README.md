# NETKit v1.5 - 网络工程师专用工具

## 📖 项目简介

NETKit 是一款专为网络工程师设计的综合性网络管理工具，提供IP配置、网络扫描、子网计算、路由跟踪和路由管理等功能。

## ✨ 主要功能

- 🌐 **IP配置管理** - 快速配置网络适配器IP地址、子网掩码、网关和DNS
- 🔍 **网络扫描** - Ping测试和网络连通性检查
- 🧮 **子网计算** - 子网划分和VLSM计算
- 🛣️ **路由跟踪** - 网络路径追踪和延迟分析  
- 🚦 **路由管理** - 静态路由添加、删除和管理

## 📦 系统要求

- Windows 10/11
- .NET 9.0 Runtime
- 管理员权限（用于网络配置功能）

## 🚀 快速开始

1. 下载最新版本的 NETKit
2. 以管理员身份运行程序
3. 选择相应的功能标签页开始使用
