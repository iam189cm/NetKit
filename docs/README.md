# NETKit v1.2

**网络工程师专业工具箱** - 集成IP配置、网络扫描、子网计算于一体的专业网络管理工具

[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Version](https://img.shields.io/badge/Version-1.2.0-orange.svg)](docs/版本管理.md)

## 🚀 功能特性

### 🖥️ IP地址配置管理
- **智能网卡识别** - 自动检测并显示所有可用网络适配器
- **快速IP配置** - 一键设置静态IP地址、子网掩码、网关和DNS
- **DHCP快速切换** - 瞬间切换到自动获取IP模式
- **实时状态监控** - 显示网卡详细信息和配置状态
- **权限智能管理** - 自动检测和提升管理员权限

### 🏓 网络扫描工具
- **可视化网格扫描** - 直观的网格界面显示IP扫描结果
- **高性能并发扫描** - 多线程并发ping，快速完成网段扫描
- **智能范围识别** - 自动识别网段，支持自定义扫描范围
- **实时进度显示** - 动态进度条和统计信息
- **颜色状态编码** - 绿色(在线)、红色(离线)、橙色(超时)直观显示

### 📊 子网计算器
- **多格式输入支持** - 支持IP+掩码和CIDR格式输入
- **完整子网信息** - 网络地址、广播地址、主机范围、可用主机数
- **VLSM计算** - 可变长子网掩码计算和优化分配
- **子网划分工具** - 智能子网划分和地址规划
- **二进制显示** - IP地址和子网掩码的二进制表示
- **地址类型识别** - 自动识别私有/公有地址、网络类别

## 🔧 功能详解

### IP配置管理
```csharp
// 核心服务类
NetworkConfigService
├── GetNetworkAdapters()           # 获取网络适配器列表
├── ApplyStaticIPConfiguration()   # 应用静态IP配置
├── SetDHCPConfiguration()         # 设置DHCP配置
└── GetAdapterDetails()            # 获取适配器详细信息
```

**特性**:
- 智能权限检测和提升
- 实时配置状态反馈
- 支持IPv4完整配置
- 自动验证输入格式

### 网络扫描引擎
```csharp
// 扫描服务架构
NetworkScanService
├── StartScanAsync()               # 异步启动扫描
├── 并发Ping执行                    # 多线程并发处理
├── 实时进度更新                    # 事件驱动状态通知
└── 结果统计分析                    # 扫描结果汇总
```

**性能指标**:
- **标准C类网段**(254个IP): 15-30秒完成
- **并发线程数**: 可配置1-100个线程
- **内存占用**: 小于50MB
- **界面响应**: 毫秒级实时更新

### 子网计算引擎
```csharp
// 计算服务功能
SubnetCalculationService
├── CalculateSubnet()              # 基础子网计算
├── CalculateSubnetFromCIDR()      # CIDR格式计算
├── SubdivideNetwork()             # 子网划分
├── CalculateVLSM()                # VLSM计算
└── IsIPInSubnet()                 # IP包含检查
```

**支持功能**:
- IPv4子网计算 (支持/8到/30)
- CIDR和传统掩码格式
- VLSM优化分配算法
- 二进制地址显示
- 网络类别自动识别

## 🚀 快速开始

### 系统要求
- **操作系统**: Windows 10/11 (x64)
- **运行时**: .NET 9.0 Runtime (自包含版本无需安装)
- **权限**: 管理员权限 (用于网络配置)
- **内存**: 最低 100MB 可用内存

### 安装运行

#### 方式一: 直接运行 (推荐)
1. 下载最新版本的 `NETKit.exe`
2. 右键选择"以管理员身份运行"
3. 允许UAC权限提升提示

#### 方式二: 从源码构建
```bash
# 克隆项目
git clone https://github.com/iam189cm/NETKit.git
cd NETKit

# 编译项目
cd src/NETKit
dotnet build

# 运行程序
dotnet run

# 或发布单文件版本
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
```

### 快速构建脚本
```bash
# 使用项目提供的构建脚本
./build-release.bat
```

## 📋 使用指南

### IP配置操作
1. **选择网卡** - 从下拉列表选择要配置的网络适配器
2. **输入配置** - 填写IP地址、子网掩码、网关、DNS
3. **应用配置** - 点击"应用配置"按钮
4. **验证结果** - 查看状态栏确认配置成功

### 网络扫描操作
1. **设置范围** - 输入要扫描的IP网段
2. **配置参数** - 设置超时时间和线程数
3. **开始扫描** - 点击"开始扫描"按钮
4. **查看结果** - 在网格中查看实时扫描结果

### 子网计算操作
1. **输入网络** - 输入IP地址和子网掩码(或CIDR格式)
2. **查看结果** - 自动显示网络地址、广播地址等信息
3. **高级功能** - 使用子网划分、VLSM计算等功能

## ⚠️ 重要提示

### 权限要求
- **管理员权限**: 修改网络配置需要管理员权限
- **UAC提升**: 程序会自动请求权限提升
- **防火墙**: 网络扫描功能可能被防火墙拦截

### 安全注意事项
- 所有网络操作都经过严格验证
- 输入数据格式检查防止错误配置
- 操作日志记录便于审计追踪
- 不会收集或传输任何用户数据

### 兼容性说明
- 支持所有标准网络适配器
- 兼容虚拟网卡和VPN适配器
- 支持IPv4网络配置
- 暂不支持IPv6配置
