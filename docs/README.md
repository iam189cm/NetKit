# NETKit

网络工程师工具箱 - 专业的网络配置管理工具。

## 技术栈

- **开发语言**: C#
- **UI框架**: Windows Forms
- **目标平台**: .NET 9.0
- **开发环境**: Visual Studio Code
- **系统要求**: Windows 10/11
- **架构模式**: 分层架构、模块化设计

## 功能特点

### MVP版本 (v1.0)
- 🖥️ **网卡选择** - 显示并选择可用的网络适配器
- ⚡ **快速IP配置** - 一键设置静态IP地址、子网掩码、网关和DNS
- 🔄 **DHCP切换** - 快速切换到自动获取IP模式
- 📊 **状态反馈** - 实时显示配置操作结果
- 🔒 **权限管理** - 智能检测和提升管理员权限

### 计划功能 (后续版本)
- 🏓 **快速Ping测试** - 图形化网段扫描和连通性测试
- 📝 **操作日志** - 记录所有配置操作和结果
- 💾 **配置保存** - 保存和加载常用IP配置
- 🌐 **批量配置** - 支持多网卡批量配置
- 📈 **网络监控** - 实时监控网络状态

## 快速Ping测试功能设计

### 功能概述
快速Ping测试是NETKit的核心扩展功能，旨在为网络工程师提供直观、高效的网段扫描工具。该功能采用图形化网格显示方式，能够快速识别网段内的活跃设备。

### 设计理念
- **图形化展示**: 采用网格布局，每个IP用彩色方块表示，状态一目了然
- **高效扫描**: 多线程并发ping，大幅提升扫描速度
- **智能识别**: 自动识别网段，支持自定义扫描范围
- **专业工具**: 面向网络工程师的专业需求设计

### 技术实现架构

#### 核心组件设计
```
src/NETKit/Core/Services/
├── NetworkScanService.cs         # 网络扫描核心服务
├── PingExecutionService.cs       # Ping执行服务
└── ScanResultManager.cs          # 扫描结果管理

src/NETKit/Core/Models/
├── NetworkScanResult.cs          # 扫描结果模型
├── IPScanItem.cs                 # 单个IP扫描项
└── ScanConfiguration.cs          # 扫描配置模型

src/NETKit/UI/Controls/
├── IPGridControl.cs              # IP网格显示控件
├── ScanControlPanel.cs           # 扫描控制面板
└── ScanStatisticsPanel.cs        # 统计信息面板
```

#### 关键技术特性
1. **并发扫描引擎**
   - 使用Task.WhenAll实现并发ping
   - SemaphoreSlim控制并发数量
   - CancellationToken支持中途取消

2. **自定义网格控件**
   - 继承UserControl实现自定义绘制
   - 支持鼠标交互和键盘操作
   - 响应式布局适应窗口大小

3. **实时状态更新**
   - 异步更新UI避免界面卡顿
   - 批量更新减少重绘频率
   - 进度条平滑动画效果

## 开发环境配置

### 必需组件
1. **.NET 9.0 SDK** - 从 https://dotnet.microsoft.com/download 下载
2. **Visual Studio Code** - 代码编辑器
3. **C# Dev Kit扩展** - VS Code中的C#开发支持

### 编译运行
```bash
# 进入项目目录
cd src/NETKit

# 编译项目
dotnet build

# 运行程序
dotnet run
```

### 打包发布
```bash
# 使用构建脚本发布
./build-release.bat

# 或手动发布为单文件exe
cd src/NETKit
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -o ../../output/Publish
```
